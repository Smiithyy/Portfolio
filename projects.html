Open-World Creature Encounter System

Introducing the Open-World Creature Encounter System, a flexible, modular foundation designed to power the exploration and encounter mechanics of your Pokémon-style game. Developed in Unity using C#, this project provides a fully functional backbone for seamless open-world exploration and creature encounters, ready to be integrated into any Pokémon clone in development. With a focus on ease of use and scalability, this system offers a robust solution for developers looking to implement key gameplay features such as wild creature encounters, turn-based combat mechanics, and an open-world exploration framework. Whether you're building a creature collection game from the ground up or looking to streamline the integration of exploration features into your existing project, this system provides the tools you need.

    Key Features:

  • Open-World Exploration: The system supports free roaming and dynamic environment generation, with customizable terrain types and regions for rich, expansive worlds.
  • Creature Encounter Mechanics: Automatically trigger wild creature encounters as players explore the world. Encounters are based on proximity, environment, and chance, offering a dynamic gameplay experience.
  • Turn-Based Combat: The project includes a base turn-based combat system, where players and wild creatures take turns performing actions. The system can be expanded to include creature abilities, type advantages, and status effects.
  • Modular and Scalable: Designed to be easily dropped into any Unity-based Pokémon clone in development. The core mechanics are highly customizable, allowing for smooth integration into your game’s evolving features.
  • Simple Creature System: Includes a basic creature system that supports attributes, health, experience points, and level progression. This can be easily extended to include custom creature stats, abilities, and evolution chains.
  • Event-Driven: Designed with flexibility in mind, the system allows you to add custom events triggered by specific in-game conditions, such as special encounters or story-driven battles.
  • C# Scripts and Documentation: All scripts are written in C# and come with clear documentation, making it easy to modify and integrate into your existing codebase.

    Perfect For:

  • Developers working on a creature collection game who want a ready-to-use framework for exploration and encounter mechanics.
  • Teams looking for a solid foundation that can be easily extended with features like creature training, evolution, and multiplayer.
  • Indie developers seeking a quick and efficient way to implement key Pokémon-style mechanics without starting from scratch.
The Open-World Creature Encounter System provides the perfect starting point for any creature collection game, allowing you to focus on crafting the unique aspects of your world, creatures, and story while leveraging the core systems for exploration and combat.

Voice Controlled Island Defence

In this exciting island defense game, you are tasked with defending your island from waves of attacking pirate ships. Using cutting-edge voice recognition, you control a powerful cannon to fend off the invaders. Aim, fire, and reload your cannon entirely through voice commands, making for a unique and immersive experience. As you progress, the pirate ships grow more aggressive, and the waves become increasingly difficult, challenging your reflexes and strategy. Created in Unity using C#, this game combines dynamic gameplay with intuitive voice controls to keep you on your toes. How long can you survive the pirate assault? The longer you last, the tougher the battle becomes—are you ready to defend your island?

Voice commands used in gameplay demo:
Fire, Turn Right, Fire, Full Left, Fire, Turn Right, Fire, Reload, Turn Right, Fire, Full Left, Fire, Turn Right


Souls-Like Combat System (Work in Progress)

This Unreal Engine 5 project features a fully functional Souls-like combat system, developed entirely using UE5's Blueprints Visual Scripting. Designed for developers looking to implement challenging, strategic combat mechanics without deep coding knowledge, this system offers a fluid and engaging player experience. With a focus on action-packed combat and resource management, this scalable system is ideal for building immersive action RPGs, dungeon crawlers, or other combat-focused games.

    Key Features:

  • Attack Combos: Players can execute a variety of attack combinations by chaining light and heavy strikes. The combo system is easy to expand or modify, allowing developers to create unique combat styles for different player characters or enemies.
  • Health and Stamina Systems: The project features a well-balanced health and stamina management system, requiring players to carefully balance offensive and defensive actions. Stamina depletes as players sprint, dodge, or attack, adding an extra layer of strategy to the combat flow.
  • Sprinting: Sprinting mechanics are integrated into the player movement, enabling rapid movement during exploration or to close gaps in combat. The stamina system is linked to sprinting, making resource management essential for success in longer encounters.
  • Dodge-Rolls: A responsive dodge-roll mechanic allows players to evade enemy attacks, providing a critical defensive option in high-stakes combat. The roll can be seamlessly integrated with attack combos, allowing players to chain defensive and offensive actions fluidly.
  • Jumping: The jumping system adds verticality to both movement and combat. Players can leap over obstacles, avoid certain enemy attacks, and perform aerial combat actions. This feature enhances the combat depth, providing more tactical options during encounters.

Scalability:

The system is designed to be scalable and flexible, easily integrated into projects of any size. It provides a foundation for combat mechanics that can be expanded with additional moves, abilities, or complex AI behavior. Developers can modify or extend the core features using UE5's Blueprint system, tailoring the combat flow and mechanics to fit specific gameplay needs, whether for smaller indie games or larger-scale RPGs. Its modular design makes it simple to tweak individual systems (such as combos, stamina, or dodging) without disrupting other elements, providing full customization based on game requirements.

Built Entirely in Blueprints:

This project leverages Unreal Engine’s powerful Blueprint Visual Scripting system, ensuring a user-friendly, no-code solution that is both accessible and efficient. By using Blueprints, developers can easily understand, adjust, and enhance the combat mechanics, without needing deep programming knowledge. The system is optimized for smooth performance and easy integration, with clear modular components that can be adjusted or extended to fit any game project.

This Souls-like combat system is a robust and versatile framework for developers looking to create a challenging and rewarding combat experience in Unreal Engine 5, while also offering the flexibility to scale and customize based on project needs
